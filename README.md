# STX4Good

**Revolutionizing Charitable Giving Through Blockchain Technology**

STX4Good is an innovative smart contract platform built on the Stacks blockchain that transforms charitable giving by creating transparent, incentivized, and impactful donation experiences. Our platform bridges the gap between donors and charitable causes while providing tangible rewards for philanthropic contributions.

##  Vision

To democratize philanthropy by making charitable giving more transparent, rewarding, and accessible to everyone through blockchain technology, fostering a global community of impact-driven benefactors.

##  Core Features

###  **Transparent Charitable Contributions**
- Make direct STX contributions to verified charitable causes
- Real-time tracking of all charitable activities on-chain
- Immutable ledger of philanthropic impact

###  **Impact Reward System**
- Earn STX4Good Impact Tokens proportional to contributions
- Build charitable giving streaks for enhanced rewards
- Unlock exclusive impact bonus rewards for dedicated benefactors

###  **Benefactor Profile Management**
- Comprehensive impact metrics tracking
- Lifetime contribution analytics
- Charitable giving streak monitoring
- Personal philanthropy dashboard

###  **Platform Governance**
- Administrative controls for platform maintenance
- Configurable minimum contribution thresholds
- Secure fund management and withdrawal capabilities

##  Technical Architecture

### Smart Contract Components

- **Benefactor Impact Profiles**: Comprehensive tracking of donor activities and metrics
- **Charitable Contribution Registry**: Immutable record of all platform donations
- **STX4Good Impact Tokens**: Fungible token rewards for charitable contributions
- **Administrative Controls**: Platform management and security features

### Key Functions

#### For Benefactors
- `contribute-to-charity`: Make charitable contributions with optional cause specification
- `claim-impact-bonus-rewards`: Redeem bonus tokens for qualified donors
- `get-benefactor-impact-summary`: View personal philanthropic metrics

#### For Administrators
- `configure-minimum-contribution-threshold`: Set minimum donation requirements
- `toggle-platform-maintenance-mode`: Enable/disable platform operations
- `withdraw-platform-funds`: Manage platform treasury

#### Platform Analytics
- `get-platform-impact-metrics`: Access comprehensive platform statistics
- `get-charitable-contribution-details`: Retrieve specific donation records

##  Getting Started

### Prerequisites
- Stacks wallet (Hiro Wallet, Xverse, etc.)
- STX tokens for contributions
- Access to Stacks testnet or mainnet

### Installation & Deployment

1. **Clone the repository**
   ```bash
   git clone https://github.com/fhay437/STX4Good.git
   cd STX4Good
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Deploy to Stacks**
   ```bash
   clarinet deploy --network testnet
   ```

### Making Your First Contribution

1. Connect your Stacks wallet to the platform
2. Select a charitable cause or leave blank for general contributions
3. Enter your desired contribution amount (minimum 1 STX)
4. Confirm the transaction and receive your Impact Tokens
5. Track your philanthropic journey through your benefactor profile

##  Impact Bonus System

### Eligibility Requirements
- Minimum lifetime contributions: 10 STX
- First-time bonus claimants only
- Active benefactor status

### Bonus Calculation
- Bonus tokens = Total lifetime contributions ÷ 10
- Example: 50 STX contributed = 5 bonus impact tokens

##  Platform Metrics

Our platform tracks comprehensive analytics:
- **Total Charitable Contributions**: Sum of all platform donations
- **Active Benefactor Count**: Number of unique contributors
- **Contribution Streaks**: Consecutive daily giving patterns
- **Impact Token Distribution**: Token reward allocation metrics
